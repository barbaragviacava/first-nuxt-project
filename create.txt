Criação do projeto Laravel:

cd backend 
docker-compose up -d
docker exec -it trum-back composer create-project laravel/laravel app
docker exec -it trum-back artisan migrate --seed

Testes Laravel:
composer require pestphp/pest-plugin-laravel --dev
php artisan pest:install

crie o arquivo chamado database/test.sqlite
altere o nome da connection de "sqlite" para "testing", o nome do database para "test.sqlite" e retire a env do DB_DATABASE no arquivo config/database.php
rode o comando artisan config:clear
descomente as linhas do arquivo phpunit.xml:
	<server name="DB_CONNECTION" value="testing"/>
	<server name="DB_DATABASE" value=":memory:"/>
	
Rodar as migrations no banco de teste: docker exec -it trum-back artisan migrate --database=testing

Para rodar os testes: docker exec -it trum-back ./vendor/bin/pest

Criado do projeto Nuxt:

comment lines 9, 10, 15 of Dockerfile
docker-compose up -d --build
yarn create nuxt-app app
uncomment lines of Dockerfile
docker-compose up --build

-------------------------------------------

Criando um teste novo:
docker exec -it trum-back artisan pest:test CAMINHO/CLASSETest

Criando uma nova model com migration (m), factory (f), controller (c) e resource (r):
docker exec -it trum-back artisan make:model -mfcr NOME_MODEL_SINGULAR

Criando um resource:
docker exec -it trum-back artisan make:resource MODELResource

Criando uma collection:
docker exec -it trum-back artisan make:resource MODELCollection
